Sub-Chapter 4. Claimant Biography LTS Screen 

a. General

The Claimant Biography screen will be auto-filled if a claimant’s record is in VADIR (VA/DoD Identity Repository).  If there is no record in VADIR, you will need to manually enter the claimant’s biographical information. 

b. New Record—Claimant Biography Screen

When creating a new chapter 33 record using the Claimant Biography screen, all fields with an asterisk are required. Pre-populated fields such as Social Security Number cannot be edited. Remember to click the Next button to save. If you click the Previous button or try to navigate back before saving, your changes will be lost. You will see the following fields, broken into 3 subsections, on the screen: 

c. Claimant

  • SSN. Social Security Number
  • Station ID. User information
  • File Number. Unique identifier
  • POA Code: Power of Attorney
  • 6X Date. This is only to be filled in if the claimant has used this before.
  • Sensitivity. This indicates the claimant’s level of sensitivity. For example, a U.S. senator may have a high level of sensitivity, so his or her record should be processed on a priority basis.

 d. Personal

  • First Name
  • Middle Name
  • Last Name
  • Suffix
  • Gender
  • Date of Birth 

e. Contact Information

  • Phone Number
  • Location (foreign, domestic, military)
  • E-mail Address
  • Address (physical address)

 

A Comments field is also featured on the Claimant Biography screen. This field is optional. 

Click the Next button to save and proceed to the Work Product History screen.

 NOTE:

When the Claimant Biography has been successfully created, a read-only banner at the top of each subsequent screen in LTS will display the Claimant File#, Claimant Name, and 6X Availability.
