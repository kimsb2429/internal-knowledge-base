Sub-Chapter 8. Enrollment: Correspondence - LTS Data Entry Rules  

This section gives you step-by-step directions on how to add an enrollment for a Correspondence training. It covers the following:

8.01 Add Correspondence Enrollment

8.02 View Correspondence Certifications

8.03 Correct a Correspondence Enrollment

 

8.01 Add Correspondence Enrollment

a. Enrollment

After the Entitlement and Kickers screen has been completed, you will navigate to the Enrollment screen. Click the Add Enrollment button to add a Correspondence Enrollment. This will cause the Add Correspondence Enrollment pop-up window to appear. All fields with an asterisk are required.

From the Add Correspondence Enrollment pop-up window, select the Training Type Correspondence from the drop-down box.

b. Enter the Begin Date, which is the effective date of the award line and must be on or after 10/01/2011. In our example, we entered 12/13/2011 for the Begin Date.

c. Enter the End Date, which is the last day of the program and must be later than or equal to the Begin Date. This field is not required. In our example, we entered 1/10/2012 for the End Date.

d. Enter the 8-digit Facility Code. The Facility Code populates the Facility Name, which will automatically display next to the Facility Code. If you do not know the facility code, search the BDN [Hyperlink to Benefits Delivery Network (BDN)].

e. Enter the 3-digit Program Code as identified in Web Enabled Approval Management System (WEAMS) or M22-4.

f. Enter the Program Name as identified on VA Form 22-1999 and approved in WEAMS.

g. Enter the Approved Lessons, Submitted Lessons, and Non-Reimbursable Lessons information in these fields.

h. Enter the Approved Rate and the Submitted Rate amounts. The Approved Rate must be greater than zero. The Submitted Rate cannot be more than the Approved Rate.

Click the Save button to return to the Enrollment screen.

 

8.02 View Correspondence Certifications

a. View Certification

Once you have added an enrollment, you will now see the new Correspondence Training enrollment on the Enrollment claim screen. The View Certs button next to each enrollment item should now be activated.

Click the View Certs button. This will display the View Correspondence Certs pop-up window. 

b. The View Flight Certification header displays the fields Begin Date, End Date, Facility Code, Facility Name, Approved Lessons, Program Code, Program Name, and Non-Reimbursable Lessons.

Below the header section is the Correspondence Certification table with the following columns:

  • Certified Through Date

  • Total Lessons to Date

c. The Certified Through Date cannot be earlier than the Begin Date or later than the End Date.

The modal action buttons on this pop-up window are Add, Edit, Delete, Save, and Cancel.

  • The Add button displays a new empty row at the top of the table where the user can enter a new certification.

  • The Edit button opens all certifications for modification.

  • When the box for a selected certificate is checked, the Delete button becomes active and allows you to remove a selected row from the table. Multiple certifications can be deleted at a single time.

Click Save to save the data and return to the Enrollment screen.

 

8.03 Correct a Correspondence Enrollment

a. Correction

If a Correspondence enrollment must be edited, click the Correction button in the expanded view of the Enrollment screen. 

When you click the Correction button, the Correct Correspondence Enrollment pop-up window will display. When all corrections and edits have been made, use the Save button to return to the Enrollment screen.
