Sub-Chapter 9. Enrollment: On-the-Job (OJT)/Apprenticeship - LTS Data Entry Rules

Sub-Chapter 9. Enrollment: On-the-Job (OJT)/Apprenticeship - LTS Data Entry Rules 

This section gives you step-by-step directions on how to add a NCD (Non-college degree) that is not from an institution of higher learning (IHL) to the enrollment screen in the LTS. This section will cover the following:

9.01 Add On-the-Job-Training (OJT)/Apprenticeship Enrollment

9.02 View OJT Certification

 

9.01 Add On-the-Job-Training Enrollment

a. After the Entitlement and Kickers screen has been completed, you will navigate to the Enrollment screen.

b. Click the Add Enrollment button to add an On-the-Job Training (OJT) or Apprenticeship enrollment.

c. From the Add Enrollment pop-up window, select the On-the-Job Training or Apprenticeship from the Training Type drop-down box.

NOTE: The fields are the same for adding an enrollment for On-the-Job Training (OJT) or Apprenticeship. In the example below, we will follow an enrollment for On-the-Job Training.

 

This will bring up another pop-up window, the Add On-the-Job Training Enrollment.

d. Enter the Program Begin Date.

e. Enter the Enrollment Begin Date. This date must be on or after 10/01/2011.

f. Enter the End Date. This date must be on or after the Program Begin Date.

g. Enter the 8-digit Facility Code. Once the code is entered, the name will automatically display next to the Facility Code. If you don't know the Facility Code, search the BDN [Hyperlink to BDN].

h. Enter the 3-digit Program Code and Program Name as identified in WEAMS or M22-4. Program Code is not a required field. However, it should be entered for each enrollment.

i. Enter the Approved Hours, which indicate the total number of hours approved for the program. Approved Hours must be greater than 0.

j. Enter the Hours/Week, which indicates the number of hours per week approved for the program. Hours/Week must be greater than 0 and cannot exceed 50.

k. Enter the Starting Step in which the claimant has entered or reentered training. Options are 1, 2, 3, 4, or 5. The default value in this field is calculated using the dates entered in the Program Begin Date and Enrollment Begin Date fields:

  • Days 0–179, then display 1

  • Days 180–359, then display 2

  • Days 360–539, then display 3

  • Days 540–719, then display 4

  • Days 720 or greater, then display 5

l. Enter the Elapsed Time, which is the amount of time credited to a claimant for prior training in that starting step. The format is MM-DD. The MM field format is as follows:

  • For steps 1–4, only months 00–05 are permitted.

  • For step 5, months 00–12 are permitted.

For the DD field, 00–29 are permitted.

Click the Save button to save the On-the-Job Training enrollment.

9.02 View OJT Certification

a. View Certs

Once you have added the enrollment, you will now see the new OJT Enrollment on the Enrollment table. Next to each enrollment item, the View Certs button should be activated.

Click the View Certs button. This will display the View On-The-Job Training Certification pop-up window.

b. Across the top of the View On-The-Job Training Certification window you will see the following fields displayed:

  • Enrollment Begin Date

  • Program Begin Date

  • End Date

  • Facility Code

  • Facility Name

  • Program Code

  • Program Name

The View Certification window contains two tables:

  • Certification Hour Table

  • Step Table

c. The Edit button opens all previously entered certifications for revision. The button also applies to any additions that need to be made to the program.

d. The Certification Hour Table is sorted in descending chronological order by year and contains the following columns:

  • Header row—displays 3-letter abbreviation of each of the months of the year, January to December.

  • Year row—Located below the header row and displays each year in which the program takes place, including the current year.

e. The Step Table is sorted in descending chronological order by Begin Date and displays the following information:

  • The Step column level of claimants training from the award record.

  • Steps are listed 1 through 5, with 5 in the first row and 1 in the last. The lowest step to display is the step number from the Starting Step field. The lowest step may not be 1.

f. The Begin Date column for the lowest step is the Enrollment Begin Date. For subsequent steps, add one day to the End Date of the Previous Step.

g. The End Date column for the lowest step is calculated by using the values in the Elapsed Time fields. To calculate the End Date, first convert the Elapsed Time to a total number of days.

  • Months (MM)

    • 00: 0 days

    • 01: 30 Days

    • 02: 60 days

    • 03: 90 days

    • 04: 120 days

    • 05: 150 days

  • Days (DD) are equal to the number of days.

  • Subtract this total value from 180.

  • Add the difference to the Begin Date to determine the End Date.

  • For subsequent steps, add 6 months to the Begin Date.

  • The last End Date in this table is equal to the enrollment End Date.

h. The Award column displays the housing rate associated with the ZIP Code of the facility multiplied by the percentage associated with the Step, which is displayed in the Percentage column.

  • 100% for Step 1

  • 80% for Step 2

  • 60% for Step 3

  • 40% for Step 4

  • 20% for Step 5
