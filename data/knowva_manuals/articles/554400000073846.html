Sub-Chapter 8. Enrollment LTS Screen 

a. General

Once Entitlement, Kickers, and any transferred benefits have been entered, you will be navigated to the Enrollment screen. The Enrollment screen is utilized to enter school enrollment(s) for an eligible individual. 

 

Click the Add Enrollment button midway down the right-hand side of the page.

 

b. Add Enrollment Pop-up Window

Select a training type from the dropdown list:

  • Graduate or Advanced Professional
  • Undergraduate College Degree
  • Non-College Degree
  • Flight
  • On-the-Job Training
  • Apprenticeship
  • Correspondence

 c. Add Undergraduate College Degree Pop-up Window

The Add Undergraduate College Degree pop-up window allows you to add enrollment information provided by the school enrollment information submitted through VA-ONCE. 

This pop-up window includes 16 data fields:

  • Training Type
  • Begin Date
  • End Date
  • Program Name
  • Hour Type (Semester, etc.)
  • FT Hours (Full-time credit hours)
  • Res Hrs (Residence hours)
  • Dist Hrs (Distance hours)
  • Vacation Days
  • Facility Code
  • Objective
  • Program Code
  • Tuition Charged
  • Fees Charged
  • Yellow Ribbon Amount
  • Interval Payment

 

Once information has been added, click the Save button.
