Sub-Chapter 11. Enrollment: Flight Training - LTS Data Entry Rules

This section gives you step-by-step directions on how to add an enrollment for a Flight Training program. This section covers the following:

11.01 Add Flight Enrollment

11.02 View Flight Certifications

11.03 Edit Flight Certification

 

11.01 Add Flight Enrollment

After the Entitlement and Kickers screen has been completed, you will navigate to the Enrollment screen. Click the Add Enrollment button to add a Flight Enrollment.

From the Add Flight Enrollment pop-up window, select the Training Type Flight from the drop-down box. All fields with an asterisk are required.

a. Enter the Begin Date, which is the effective date of the award line and must be on or after 10/01/2011.

b. Enter the End Date, which is the last day of the program and must be later than or equal to the Begin Date.

c. Enter the 8-digit Facility Code. The Facility Code populates the facility Name, which will automatically display next to the Facility Code. In our example, we entered 15001218 for the Facility Code. If you do not know the facility code, search the Benefits Delivery Network (BDN) [Hyperlink to BDN].

d. Enter the 3-digit Program Code as identified in Web Enabled Approval Management System (WEAMS) or M22-4. Program Code is a required field. In our example, we entered 341 for the Program Code.

e. Enter the Program Name as identified on VA Form 22-1999.

f. Enter the Approved Charges, Previous Charges, and Total Submitted Charges.

g. Enter the amount of Approved Hours, Previous Hours, Submitted Hours, and Approved Rate for the respective Flight types.

h. If the Taxes are Payable for this Facility checkbox is checked, taxes included are payable and can be entered on the Flight Certification enrollment screen.

i. The Med Cert drop-down options are First Class and Second Class. First Class is the default. If the checkbox for Med Cert Invalid on Begin Date is checked, the enrollment is invalid.

11.02 View Flight Certifications

Once you have added an enrollment, you will now see the new Flight Training Enrollment on the Enrollment screen. The View Certs button next to each enrollment item, should now be activated. 

a. Click the View Certs button. This will display the View Flight Certification pop-up window. 

b. The View Flight Certification header displays the fields Begin Date, End Date, Facility Code, Facility Name, Program Code, and Program Name.

The middle header row displays the Remaining Hours and Approved Rates for each flight type that was entered.

c. Click the Add button to add a Flight Certification.

d. Enter the Training Dates, Hours, and Rates and click the Save button.

e. Clicking Save will return you to the Enrollment screen. The Certificate information cannot be viewed from the Enrollment screen. If you need to edit a Flight Certification, click the View Certs button on the Enrollment screen to bring up the View Flight Certifications pop-up window.

11.03 Edit Flight Certifications

a. To edit a training period, select the checkbox for the appropriate training period. The Add, Edit, and Delete buttons for the training period will be activated.

b. Click the Edit button to change the Dual Rate for this Flight Certification. Then click the Save button.
