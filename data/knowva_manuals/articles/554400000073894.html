6.01 VA-ONCE Inbox

a. The VA-ONCE Inbox section is used to display enrollment and change in enrollment information submitted by the school via the VA Online Certification of Enrollment (VA-ONCE). 

If there are no VA-ONCE submissions for the claimant, the message “The claimant has no VA-ONCE records at this time” will be displayed.

If new information has been sent through VA-ONCE, the submissions will be displayed in the VA-ONCE INBOX section.

b. The Decline, Add, and Correct buttons will become active depending on the specific circumstances when a VA-ONCE submission is selected.

6.02 Enrollment

a. The Enrollment section will display data about schooling or training where a claimant is or has been enrolled. This is also the section where you will add new enrollment for a claimant.

If you need to correct or amend an enrollment for a claimant, the following information will be discussed in the Enrollment Correct and Add Function section:

Click Add Enrollment to start. This will display the Add Enrollment pop-up window.

 

b. Add Enrollment Pop-up Window

In the Add Enrollment pop-up window, select the Training Type from the drop-down box:

  • Graduate or Advanced Professional
  • Undergraduate College Degree Non-College Degree
  • Flight
  • On-the-Job Training
  • Apprenticeship
  • Correspondence

Once you choose the type of training, click Submit to move on, or Cancel to start over.

Clicking Submit for the type of training will display the Add Undergraduate College Degree pop-up window. 

 

c. Add Undergraduate College Degree Pop-up Window

The Add Undergraduate College Degree pop-up window allows you to add enrollment information provided by the school or to verify or correct enrollment information submitted through VA-ONCE. 

This pop-up window includes 16 data fields:

  • Training Type
  • Begin Date
  • End Date
  • Facility Code
  • Objective
  • Program Code
  • Program Name
  • Hour Type (Semester, etc.)
  • FT Hours (Full-time credit hours)
  • Res Hrs (Residence hours)
  • Dist Hrs (Distance hours)
  • Vacation Days
  • Tuition Charged
  • Fees Charged
  • Yellow Ribbon Amount
  • Interval Payment

d. Begin Date is the first day of the Term, Semester, or Quarter.

e. End Date is the last day of the term as shown on the enrollment certification. It is not the No Pay Date as used in other benefit systems.

f. Facility Code is an 8-digit code that corresponds with the institution the claimant will be attending. The field will auto-populate if you type in the facility name, or you can search the BDN [Hyperlink to interface] for the code. The Facility Name will automatically display next to the Facility Code.

g. Objective is a drop-down box. This is a required field. The available options are:

  • 011AA
  • 012AS
  • 013AD-NCE
  • 021BA
  • 022BS
  • 023BD-NEC

h. Program Code is a 3-digit code that can be as identified in WEAMS or M22-4. Program Code is not a required field. However, it should still be entered for each enrollment.

i. Program Name is the name of the type of program being pursued (for example, B.S. Computer Science).

j. Hour Type is a drop-down box that displays the type of term offered at the institution (for example, Semester).

k. FT Hrs (Full-Time Hours) represents the number of hours required for full-time training. This field auto-filled WEAMS, based on the Facility Code entered.

l. Res Hrs (Residence Hours) and Dist Hrs (Distance Hours) will accept up to 4 numeric characters. The total of Res Hrs and Dist Hrs must be greater than zero.

m. Vacation Days is not a required field and is only applicable to nonstandard periods of enrollment. Acceptable entries are 0, or 7 to 99.

n. Tuition Charged, Fees Charged, and Yellow Ribbon Amount are not required fields. Each field will accept up to 9 numeric characters with a format of 999999.99.

o. (rescinded) Effective August 1, 2011 interval payments are no longer permitted. Archive: The default is No Restrictions. Available options are:

  • No restrictions
  • Student requests no interval payment
  • Change in school with no change in program
  • Change in school and change in program
  • Summer Session/No restrictions
  • Summer Session/Student requests no interval payment
  • Summer Session/Change in school with no change in program
  • Summer Session/Change in school and change in program

Click the Save button to return to the Enrollment screen.
