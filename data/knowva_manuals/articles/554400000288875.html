II.3.D.1.a

Eligibility banner

When a chapter 35 claim has been created, a claim banner displays under the claimant banner on all tabs of the claim.

If the beneficiary is, or previously was, eligible and the most recently authorized claim was for chapter 35, the claim banner displays the following information in read-only format.

  1. Survivors’ and Dependents’ Education Assistance (Chapter 35)
  2. DEA eligibility start date:  This field displays the earliest date the beneficiary may use their chapter 35 benefits in MM/DD/YYYY format.  It is determined by the Election date entered on the Eligibility status page.
  3. Verified through date:  This field displays the most recent chapter 35 enrollment verification date in MM/DD/YYYY format.
  4. Delimiting date:  This field displays the chapter 35 delimiting date in MM/DD/YYYY format.  If the beneficiary has no delimiting date, this field will display as N/A.
  5. Original entitlement:  This field displays the total original chapter 35 entitlement the beneficiary is eligible for in MM-DD format.
  6. Entitlement used:  This field displays the total chapter 35 entitlement the beneficiary has used in MM-DD format.
  7. Entitlement remaining:  This field displays the amount of chapter 35 entitlement remaining in MM-DD format.  This does not display when entitlement exhausted.
  8. Entitlement exhaust date:  This field only displays when chapter 35 entitlement has been exhausted and shows the date of exhaustion in MM/DD/YYYY format.

Figure: Chapter 35 claim banner

If a beneficiary has used the five-month free enrollment, an additional line in the claim banner displays with the following information.

  1. 5 month free entitlement
  2. Entitlement used:  This field displays the amount of entitlement used for five-month free entitlement in MM-DD format.
  3. Entitlement remaining:  This field displays the amount of five-month free entitlement remaining in MM-DD format.  This does not display when the five-month free entitlement is exhausted.
  4. Entitlement exhaust date:  This field only displays when the five-month free entitlement has been exhausted and shows the date of exhaustion in MM/DD/YYYY format.

Figure: Chapter 35 five-month free entitlement claim banner

If the beneficiary is denied on their original claim and has not subsequently established chapter 35 eligibility, the claim banner displays the following information in read-only format.

  1. Survivors’ and Dependents’ Education Assistance (Chapter 35)
  2. VA eligibility status: This field displays Denied.
  3. Date of denial:  This field displays the date the denial was authorized in MM/DD/YYYY.

Figure: Chapter 35 claim denial banner

        2. The user can either manually type in the date or select the appropriate date from the calendar icon.
      2. Spouse is at fault:  The user selects this checkbox if the spouse of the beneficiary (i.e., the Veteran) was at fault.
      3. Remarriage date:  This field is required if a remarriage date is provided.
        1. Must be in MM/DD/YYYY format.
        2. The user can either manually type in the date or select the appropriate date from the calendar icon.

        2. The user can either manually type in the date or select the appropriate date from the calendar icon.

    6. I have established the beneficiary’s relationship to the Veteran by reviewing the relevant documents.  The user selects this checkbox to verify the relationship.

  3. If the Relationship type is Stepchild, the following information can be entered.

    2. Beneficiary’s date of birth:  This field is automatically filled in from the beneficiary’s file and cannot be edited here.  If needed, the user can edit this field on the Biographical information screen.
    3. Parents’ marriage date:  This field is required. 
      1. Must be in MM/DD/YYYY format.
      2. The user can either manually type in the date or select the appropriate date from the calendar icon.

        1. Must be in MM/DD/YYYY format.
        2. The user can either manually type in the date or select the appropriate date from the calendar icon.
      2. Veteran is providing at least 50% care for dependent:  The user selects this checkbox if applicable.
    5. Provide Veteran’s date of death:  The user selects this checkbox if the Veteran has a date of death on file.  If selected, the following fields will display.

Systematic delimiting date extensions

For chapter 35 beneficiaries whose delimiting date occurs during a term or course, the delimiting date can be extended in some circumstances.

  1. If the school operates on a semester or quarter basis, the extension is automatically granted by BM through the end of the term.  The user should review the extension and send the award for second signature authorization.
  2. If the school does not operate on a semester or quarter basis (i.e., clock hours) and more than half of the course is completed by the original delimiting date, the extension is granted until the end of the course or for 12 weeks (84 days) from the original delimiting date.  The user should review the extension and send the award for second signature authorization.

Exception: Chapter 35 spouses can have their delimiting date extended until the end of the course or for 12 weeks (84 days) from the original delimiting date even if they have not completed half of the course by the original delimiting date.

II.3.D.1.g

Multiple sources

BM allows the concurrent payment of benefits under multiple sources of chapter 35 entitlement.  Once chapter 35 eligibility has been established under one source of entitlement, the user can select the Add an additional source button to add an additional source functioning.  See M22-4, Part II, 3.C for details.  Users can add up to five total entitlement sources.

If there is reason to not pay under multiple sources, the user selects a VA eligibility status of Suspended – Elected not to use source. 

Note: If multiple entitlement sources are entered and not suspended, BM will automatically pay concurrently where possible without any additional user interaction.

Note:  Correspondence awards can only be paid under a spouse entitlement source for chapter 35.  Licensing and certification, national exam, tutorial assistance, and work study cannot be paid multiple times.

Adding multiple entitlement sources will add the corresponding information to the chapter 35 eligibility banner.

II.3.D.1.h

Navigation

Selecting the Back or Exit claim buttons at the bottom of the Eligibility screen returns the user to the beneficiary search screen (See M22-4, Part II, 3.A for details). 

Selecting the Continue button at the bottom of the Eligibility screen advances the user to the Entitlement screen (See M22-4, Part II, 3.E for details.)

Selecting the Return to top button brings the user back to the top of the Eligibility screen.

Selecting the Search beneficiaries buttons brings the user to the beneficiary search screen (See M22-4, Part II, 3.A for details).

For chapter 1606 beneficiaries whose delimiting date occurs during a term or course, the delimiting date can be extended in some circumstances.

  2. If the school does not operate on a semester or quarter basis (i.e., clock hours) and more than half of the course is completed by the original delimiting date, the extension is granted until the end of the course or for 12 weeks (84 days) from the original delimiting date.  The user should review the extension and send the award for second signature authorization.

  4. Delimiting date extensions do not apply for beneficiaries pursuing a program of Apprenticeship or On-the-job training.
