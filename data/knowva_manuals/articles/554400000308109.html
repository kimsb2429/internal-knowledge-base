College-Level Course and Objective Code.  A course code is assigned by the ELR (Education Liaison Representative) for each approved program. 

For IHL (Institutions of Higher Learning) courses, the code entered in the objective field should identify the degree level.  The course code, on the other hand, should identify the specific field of study.  For example, a BS in chemistry should have an objective code of 022 and a course code of 162.  For an MS in chemistry, the objective code should be 032 and the course code should be 162.

Use the appropriate codes from this section for institutions of higher graduate training or undergraduate training. Use the appropriate 200 series codes from this section for institutions of higher learning with nondegree courses.

Throughout the various codes, the term "N.E.C." stands for Not Elsewhere Classified.
