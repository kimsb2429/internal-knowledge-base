# Sub-Chapter 1. User Login LTS Screen

**Sub-Chapter 1. User Login LTS Screen**

**a. User Login**

You will begin at the **CH33 LTS Login** page. Each Veterans Claims Examiner (VCE) enters the following information:

* Station ID (Location # for the Regional Processing Office where you work—Buffalo (307); Muskogee (351))
* User ID
* Password

This information should have been given to all Veterans Claims Examiners during orientation. If you do not have this information, contact your supervisor.
