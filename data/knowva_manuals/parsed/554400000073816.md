# Sub-Chapter 3. Search Chapter 33 LTS Screen

**Sub-Chapter 3. Search Chapter 33 LTS Screen**

**a. Search**

The **Search Chapter 33** screen is where you will start all your claims processing. To start processing a claim, find the claimant’s record in the LTS. Begin the search by entering any of the following in the corresponding fields on the screen:

* Social security number (SSN) or File #
* Date of Birth
* Last Name, First Name

Click the **Search Chapter 33** button.

**![LTS Chapter 33 Search](https://www.knowva.ebenefits.va.gov/img/edu/XII53_Search_Chapter_33_LTS_Screen_img1.jpg)**

If no results are found in the Chapter 33 (CH33) LTS, you will see the message **No Results Found.** A **“Search VADIR”** button will appear on the right-hand side of the screen. VADIR is the VA/DoD Information Repository.

**![LTS Chapter 33 Search No Results](https://www.knowva.ebenefits.va.gov/img/edu/XII53_Search_Chapter_33_LTS_Screen_img2.jpg)**

If no results are found in VADIR, you will see the following warning message:

**b. No VADIR data**

***No association of [Claimant] in VADIR.***

If the claimant is in VADIR, all personal identifying information will auto-fill in LTS. The user will not need to key in additional information, but should compare and verify results to the application including address, phone number, e-mail address, etc.

If no matches are found in either CH33 LTS or VADIR, you should perform a new CH33 search by refining the search by adding a last name or DOB. If no matches are found, you will need to manually create a new CH33 record.

Click the **Create New Chapter 33 Record** button on the right-hand side of the screen to navigate to the **Claimant Biography** screen.

**![LTS Chapter 33 Search No Match in VADIR](https://www.knowva.ebenefits.va.gov/img/edu/XII53_Search_Chapter_33_LTS_Screen_img3.jpg)**
