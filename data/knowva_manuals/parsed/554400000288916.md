| **II.3.F.2.a**  **Adding graduate enrollments** | To add a new graduate enrollment, the user selects the **Add new enrollment** button to open the Add a new enrollment window.  The user then chooses *Graduate or advanced professional* from the **Training type** dropdown menu and selects the **Continue** button to advance to the Add graduate or advanced professional enrollment window.  Users also can use the **Cancel** or **X** button to close the window without taking any actions.  In the Add graduate or advanced professional enrollment window, users enter the following information.   1. **Begin date**:  This field is required.     1. Must be in MM/DD/YYYY format.    2. The user can either manually type in the date or select the appropriate date from the calendar icon.    3. The date cannot be more than 180 days in the future. 2. **End date**:  This field is required.     1. Must be in MM/DD/YYYY format.    2. The user can either manually type in the date or select the appropriate date from the calendar icon. 3. **Facility code**:  This field is required.  Must be in X-X-XXXX-XX or XXXXXXXX format. The user can enter with or without hyphens. 4. **School name**:  This field is not editable.  It is automatically filled in based on the entered **Facility code**. 5. **Objective**:  This field is required.  The user selects one of the following options from the dropdown menu:    1. *Master of Arts*,    2. *Master of Sciences*,    3. *Master’s degree*,    4. *Doctor of Philosophy*,    5. *Doctor’s degree*, or    6. *Postdoctoral*. 6. **Program name**:  This field is required.  The user selects one of the options from the dropdown menu that displays all Institution of Higher Learning (IHL) programs for that school.    1. The user can start typing the program name from the enrollment certification to filter the programs listed.    2. If the enrollment certification reports a non-matriculated term, the user can enter “non-matriculated” in this field to process the claim. 7. **Hours type**:  This field is required and is automatically filled based on the entered **Facility code**.  The user can adjust the selection to the following options from the dropdown menu:    1. *Semester*,    2. *Quarter*, or    3. *Clock* 8. **Undergraduate full-time hours**:  This field is required and is automatically filled in based on the **Facility code** with the corresponding value in WEAMS.  The user may adjust this value, if appropriate. 9. **Enrolled hours**:  This field is required.     1. The user enters a positive value in hours and fraction of an hour in XX.XX format.    2. This field must be greater than 00.00 hours.    3. This field is automatically filled if the enrollment certification is received from Enrollment Manager (EM) with the sum of *residence* and *distance* credit hours.  The user may adjust this value, if appropriate. 10. **Tuition & fees**:  This field is required, if provided.       1. This value must be between $0.00 and $99,999.99, inclusive.     2. If no tuition & fees value is provided on the enrollment certification, the value is $0.00.     3. Must be provided in the following situations:        1. The **Top-up** checkbox is selected,        2. The **Incarcerated** checkbox is selected,        3. The training time is less than ½ time or ¼ time, or        4. The beneficiary is on active duty during the term. 11. **Vacation period**:  These fields are required only if a vacation period of 7 days or longer is certified.  The user enters the following information.     1. **Start date**:  The user enters the start date of the vacation period in MM/DD/YYYY format.     2. **End date**:  The user enters the end date of the vacation period in MM/DD/YYYY format.     3. The user selects the **Add another vacation period** button if more than one vacation period is needed. 12. **Certified TT** (training time):  This field is required for graduate enrollments.  The user selects one of the following options from the dropdown menu:     1. *Full-time*,     2. *¾ time*,     3. *½ time*,     4. *Less than ½ time*, or     5. *¼ time*. 13. **Top-up**:  The user selects this checkbox if the certification is for a Top-up enrollment. See M22-4, Part II, 3.A.3.b for details on Top-up enrollment and kickers. 14. **Incarcerated**:  The user selects this checkbox if the certification is for an incarcerated beneficiary.   The user selects the **Add enrollment** button to save the new enrollment.  Users also can use the **Cancel** or **X** button to close the Add graduate or advancement professional enrollment window without taking any actions.  This is an image of the add graduate or advanced professional enrollment modal.  This is an image of the add graduate or advanced professional enrollment modal.  *Figure: Add graduate enrollment* |
